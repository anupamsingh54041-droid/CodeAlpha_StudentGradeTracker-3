# 🎓 Student Grade Tracker

A console-based Java application developed for the CodeAlpha Java Programming Internship.

This project helps manage student records and analyze their academic performance by calculating average marks, highest marks, and lowest marks.

---

## 🚀 Features

- Add student names and marks
- Calculate average marks
- Find highest marks
- Find lowest marks
- Generate a simple student performance report
- Easy-to-use console interface

---

## 🛠 Technologies Used

- Java
- Arrays
- Loops
- Conditional Statements
- Scanner Class

---

## 📋 How It Works

1. User enters the number of students
2. Program takes student names and marks as input
3. Marks are stored using arrays
4. The program calculates:
   - Average Marks
   - Highest Marks
   - Lowest Marks
5. Final report is displayed on the console

---

## ▶️ How to Run

1. Open the project in any Java IDE
2. Run the `Main.java` file
3. Enter student details and marks
4. View the generated report
