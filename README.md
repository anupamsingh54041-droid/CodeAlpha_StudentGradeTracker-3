# 🎓 Student Grade Tracker

This project is developed using Java for the CodeAlpha Java Programming Internship.

The Student Grade Tracker helps manage student marks and perform different calculations such as average marks, highest marks, and lowest marks.

---

## 🚀 Features

- Add student names
- Enter student marks
- Calculate average marks
- Find highest marks
- Find lowest marks
- Display complete student report

---

## 🛠 Technologies Used

- Java
- Scanner Class
- Arrays
- Loops
- Conditional Statements

---

## 📌 Project Type

Console-Based Java Application

---

## ▶ How to Run

1. Open the project in any Java IDE
2. Run the `Main.java` file
3. Enter student details and marks
4. View the generated report
